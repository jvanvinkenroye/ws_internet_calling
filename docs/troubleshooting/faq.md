# Frequently Asked Questions

## General Questions

**Q: What is a Nummernsender?**  
A: A number transmitter system that broadcasts numbers in a sequence.

**Q: Do I need a Raspberry Pi Pico?**  
A: No, the web application and API work standalone. The Pico is only needed for IoT examples.
