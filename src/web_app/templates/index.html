How to Use These Examples

  1. Download the example you want to try
  2. Open it in Thonny IDE
  3. Update the WiFi credentials (WIFI_SSID and WIFI_PASSWORD)
  4. For API consumer examples, update the API_BASE_URL with your server's IP
  5. Upload to your Raspberry Pi Pico W and run!
