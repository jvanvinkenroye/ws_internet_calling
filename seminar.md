# Seminar Nummernsender im Internet
## Exkurs Nummernsender
-> Online ein Beispiel suchen
### Automatischer Nummernsender (als Webseite)
-> Webseite in python + html implemmentieren, die Webseite gibt die Nummern zwischen 1 und 9 aus was jede sekunde wechlselt
### Maschinenlesbarer Nummernsender (als API)
-> -> Webseite in python + api implemmentieren, die Webseite gibt die Nummern zwischen 1 und 9 aus was jede sekunde wechlselt
## Vorstellung Raspberry Pi Pico
- Programmieren in Python
- Flashen wie ein USB- Drive
- Einfacher flashen über Thony
- Beispiel blink.py 
- Beispiel autostart (blink.py als main.py)
## Raspberry Pi Pico im Wlan
- Raspberry Pi Pico mit dem Wlan verbinden (connect oder not connected)
- Raspberry Pi Pico mit dem Wlan verbinden und Signal messen und auf der Console ausgeben.
- Raspberry Pi Pico mit dem Wlan verbinden und Signal messen und die stärke vom Signal in Blinkfrequenz der internen LED umsetzen. (stärker = schneller, schwächer = langserm)
## Rasbperry Pi Pico über Wlan eine Api 
- Raspberry Pi Pico über Wlan eine Api afragen und anzeigen.
- Raspberry Pi Picoe über Wlan eine Api abfragen und in Blink umsetzen.
## Raspberry Pi Pico als WLAN Access Port
- Raspberry Pi Pico stellt html-seite bereit:
	- Bezieht ip per DHCP
	- Gibt die IP auf console aus
	- Auf dem Pi läuft eine Webseite, mit der man die interne LED ein oder auschalten lann.
